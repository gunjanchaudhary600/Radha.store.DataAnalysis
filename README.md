# Radha.store.DataAnalysis
This project analyzes Radha Store Sales data using Microsoft Excel. The Goal was to clean the raw data, perform analysis using Pivot Tables, and create an interactive dashboard to visualize sales performance and business insights.
DATA CLEANING: 
Removed Duplicate Records,
Handled missing and inconsistent values,
Foormatted and structured the data for accurate analysis
DATA ANALYSIS:
Created Pivot Table to summarize sales data,
Analyzed sales based on different categories such as product, category, and time,
Identified key patterns and trends in sales performance.
DASHBOARD CREATION:
Built an interactive Excel dashboard using charts and Pivot Tables,
Used Slicers to enable dynamic filtering and easy interaction,
Visualized key matrics such as total sales, category-wise performance, and trends.
TOOLS USED: 
Microsoft Excel, Pivote Tables, Slicers, Charts & Data visualization, Data Cleaning Techniques.
KEY OUTCOMES:
The dashboard provides a clear and interactive view of Radha Store's sales data. Slicers allow quick filtering, making it easier to explore insights and support data-driven decision-making.
